# ESP8266 min config pcb

Minimalistic form for the ESP8266 12F.

Component list:
- 1x 100nf ceramic cap
- 4x 100k resistors
- 1x 10k resistor
- 1x 70mmx50mm pcb
- Headers female/male
- LAN cable wires
- 1x pushbutton (reset)
- 1x switch (programming or running mode)
